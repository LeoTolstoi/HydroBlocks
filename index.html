Background

HydroBloks is a groundbreaking approach to hydrologic modeling. For the first time ever it provides a viable path towards continental scale hydrologic modeling that provides field scale information while remaining computational efficient.

Main Features

  • Modularity
  • High Performance Computing
  • Hydrologic Similarity

Tutorial - Little Washita Catchment, Oklahoma

Let start off with installing the necessary python libraries

$ pip install numpy
$ pip install netCDF4

Download the data for the Little Washita catchment

$ wget

We can now run the model for the default settings

$ python driver.py run_model

Let's now create a set of rudimentary plots

$ python driver.py create_plots
